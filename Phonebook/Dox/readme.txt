Система словарей 09 Tray Help System представляет из себя миниатюрную
In-Memory Database вызываемую как POP-UP справка.


Непрактично, хотя и возможно, хранить в 09 Tray Help System, например, Большой
Энциклопедический Словарь (БЭС). Типичное применение 09 - это хранение
небольших (до нескольких десятков тысяч) и редко обновляемых списков.
Например, телефоны организации, толковые или двуязычные мини-словари,
всевозможные расписания и прейскуранты, таблица неправильных глаголов,
интернет-пароли, списки памятных дат (религиозных праздников), сборники
афоризмов итп.


----------------------------

Версия LITE абсолютно бесплатна для любого использования.



Ограничения версии LITE:

-  нет экспорта
-  нет запрета декомпиляции для создаваемых баз
-  Одновременная работа только с одним файлом базы.
   Если требуется работа с другой базой, необходимо закрыть
   программу, заменить файл базы на нужный и запустить
   программу вновь.
